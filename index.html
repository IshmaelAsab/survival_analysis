How to use the analysis operationally

  • Prioritize first-90-day interventions because churn intensity peaks around month 1.
  • Use contract design as a primary retention lever; one- and two-year terms dominate the strongest protective effects.
  • Flag Offer E cohorts for rapid remediation, since they show materially worse churn timing in both Cox and AFT views.
  • Bundle online security, backup, and dependents-based messaging into lifecycle campaigns to lengthen customer survival.

Deployment

Vercel-ready static app

This dashboard is intentionally dependency-light: Vercel can host it as a static site directly from this repository, while preserving links to the generated analysis artifacts under outputs/.

  1. Import the repository into Vercel.
  2. Leave the framework preset as Other.
  3. No build command is required for this static dashboard.
  4. Deploy and optionally assign a production domain.
